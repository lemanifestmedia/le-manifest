# Le Manifest — Média international indépendant

**Le Manifest** est un média français indépendant d’actualité internationale, lancé en 2021 depuis La Ciotat (France).
Le média privilégie l’analyse, le décryptage et la compréhension globale des enjeux mondiaux.
